# react-native-line-login


Let your users sigin with their LINE account.

## Installation
### - With RNPM

```bash
"react-native-line-login": "git+https://github.com/miyabiya/react-native-line-login.git"
```

```bash
yarn install
```

```bash
react-native link react-native-line-login
```
